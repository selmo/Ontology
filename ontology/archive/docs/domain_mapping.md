# data.go.kr 기반 온톨로지 재구성 계획

## 1. data.go.kr 16개 대분류

공공데이터포털(data.go.kr)의 표준 분류체계:

1. **공공행정** - 정부 조직, 지방행정, 행정서비스
2. **과학기술** - R&D, 특허, 연구개발
3. **교육** - 학교, 교육정책, 평생교육
4. **교통물류** - 도로, 대중교통, 물류
5. **국토관리** - 토지, 건축, 도시계획
6. **농축수산** - 농업, 축산, 수산업
7. **문화관광** - 문화재, 관광지, 공연
8. **법률** - 법령, 판례, 법률서비스
9. **보건의료** - 의료기관, 질병, 건강
10. **사회복지** - 복지시설, 사회보장, 취약계층
11. **산업고용** - 기업, 일자리, 산업통계
12. **식품건강** - 식품안전, 영양, 위생
13. **재난안전** - 재난관리, 안전시설, 비상대응
14. **재정금융** - 예산, 세금, 금융정책
15. **통일외교안보** - 외교, 통일정책, 안보
16. **환경기상** - 환경보전, 기후, 날씨

## 4. 재구성 방안

### 옵션 A: 완전 전환 (16개 도메인)

**장점:**
- 공공데이터포털 표준 준수
- 향후 공공데이터 연계 용이
- 포괄적인 분류 체계

**단점:**
- 현재 문서가 없는 도메인 다수
- 대규모 재작업 필요
- 커머스 등 민간 영역 분류 부재

### 옵션 B: 하이브리드 (현재 + data.go.kr 확장)

**장점:**
- 기존 구조 유지
- 필요한 분류만 추가
- 점진적 확장 가능

**단점:**
- 이중 체계 혼재 가능성
- 표준 준수 불완전

### 옵션 C: 선별적 적용 (10개 도메인)

현재 문서가 있거나 향후 확장 가능성이 높은 도메인만 선택:

1. **01 공공행정** (현 공공) ← public/
2. **02 교육** (신규)
3. **03 보건의료** (현 의료) ← medical/
4. **04 사회복지** (현 사회)
5. **05 법률** (현 법률) ← law/
6. **06 재정금융** (현 금융) ← finance/
7. **07 산업고용** (현 산업 + 커머스) ← commerce/
8. **08 재난안전** (신규)
9. **09 문화관광** (신규)
10. **10 환경기상** (신규)

**장점:**
- 표준 준수하면서 실용적
- 기존 데이터 활용
- 확장성 확보

**단점:**
- 일부 표준 분류 누락

```

C01010001 = 도메인01 + 중분류01 + 일련번호0001
```
